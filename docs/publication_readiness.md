# Publication Readiness

TLMpy is not JOSS-ready at v0.1.

1. The repo must be public for more than six months with iterative development.
2. Demonstrated research use is required; aspirational future-use statements are insufficient.
3. A scalar demonstrator plus FTCS reference risks being read as educational or reinvention next to Meep and `fdtd`.
4. The path is to release v0.1 publicly, develop iteratively in the open, reach a v0.2 scholarly differentiator, accumulate real use, then submit.
