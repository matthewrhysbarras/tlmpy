# Summary

Draft, not submission-ready at v0.1.

TLMpy is an experimental Python package for scalar 2D Transmission Line Matrix style time-domain simulation with a finite-difference diffusion reference solver and early research prototypes for parabolic TLM diffusion studies.

# Statement of Need

TLM offers a port-scattering view of wave propagation that differs from generic finite-difference and FDTD formulations. TLMpy positions this implementation against Meep, the Python `fdtd` package, and established TLM codes by emphasizing a compact Python-first scalar demonstrator with explicit validation tests. The diffusion tools are validation and research case-study infrastructure; the experimental parabolic TLM estimator work is not yet submission-ready.
